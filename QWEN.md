# 项目概述

- 每次占卜，都将结果以markdown格式文件，存在divination_records文件夹下。
- 这是一个关于**《周易》和《高岛易断》**的数字化资料整理与查询工具项目。项目核心目的是将传统易学经典文献进行数字化处理，建立结构化的数据库，并提供便捷的卦象查询功能。

## 项目内容

### 核心资料
- **《周易》六十四卦**: 包含原文HTML文件(64卦)和配套图片(600+)
- **《高岛易断》**: 日本易学家高岛嘉右卫门的易学著作，包含占断破解详解
- **结构化数据库**: CSV格式存储卦象、爻辞、上下卦组合等核心数据

### 技术实现
- **Python查询脚本**: 通过输入上卦编号、下卦编号、动爻编号，快速查询卦象信息
- **多格式转换**: EPUB → HTML/Markdown，便于不同场景使用
- **数据关联**: 建立卦象与在线资源的链接映射

## 使用方法

### 运行查询脚本
```bash
cd python
python 07.py
```
按提示输入:
1. 上卦编号(1-8)
2. 下卦编号(1-8)
3. 动爻编号(1-6)

输出: 上下卦信息、卦象、释义、相关链接、动爻

### 数字起卦法 (用户偏好)
用户采用数字起卦法进行占卦：
1. 心中想一个问题
2. 随意报两个数字：第一个数÷8余数定上卦，第二个数÷8余数定下卦
3. 再报一个数字：÷6余数定动爻（余数为0则取6）

> **注意**: 用户偏好使用报数起卦法，认为比程序随机生成更有诚意和关联性。

## 项目特点

1. **资料完整**: 涵盖周易原文和高岛易断占断详解
2. **结构化存储**: CSV数据库便于程序化查询和扩展
3. **多格式支持**: HTML/Markdown/EPUB多种格式并存
4. **在线资源链接**: 每条数据关联原文出处(zhouyi.cc)和详细解读(Gitee Markdown)
5. **交互查询**: Python脚本实现快速卦象检索
6. **占卦记录**: divination_records/ 目录保存每次占卦记录
