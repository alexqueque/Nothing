# 使用方法

 传统占卜获取卦象的方法

    如果你想知道如何传统地得到一个卦象,常见方法有:

    铜钱摇卦法(推荐)
     1. 准备3枚铜钱(或硬币)
     2. 双手合扣铜钱,心中默念所问之事
     3. 摇动后抛出,记录结果:
        - 1个背+2个字 = 少阳(记为7)
        - 2个背+1个字 = 少阴(记为8)
        - 3个背+0个字 = 老阳(记为9, 动爻)
        - 0个背+3个字 = 老阴(记为6, 动爻)
     4. 重复6次,从下到上记录,得到6爻
     5. 根据6爻的阴阳组合确定上下卦
     6. 记录哪些爻是动爻(9或6)

    数字起卦法
     1. 心中想一个问题
     2. 随意报出两个数字(如第一个数÷8的余数定上卦,第二个数÷8的余数定下卦)
     3. 再报一个数字(÷6的余数定动爻,余数为0则取6)
