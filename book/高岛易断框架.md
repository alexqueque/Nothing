## 一、前置部分

### 1. 前言与序跋
- **作者序**: 高岛吞象自述。核心论点——「易」是通神之书，强调"至诚无息"的占卜心法，批评后世学者拘泥文义、废其实用。
- **栗本锄云序**: 点明高岛易学是七年狱中实践之结晶，非纯理论推演。
- **副岛种臣序**: 推崇高岛以易占决实际事务的能力。
- **译者序**: 王治本介绍翻译缘起，强调此书"照之于事物之实际"的特色。
- **编校者注**: 曾子恒的现代校注说明。

### 2. 高岛吞象传略
高岛一生关键节点：
- 1832年生于江户，继承父业
- 1861年因货币兑换入狱
- 狱中发现《易经》，每日课一卦，七月卒业
- 以易占逃过狱中生死劫（《履》之《井》卦）
- 1865年获释，创业成功（木材、旅馆、铁路、煤气）
- 1877年引退，专攻易学研究
- 1914年去世

### 3. 高岛吞象易经筮法揭秘
**核心占卜方法——"至诚无息"法：**
- 用50根蓍草（或50枚硬币），分二、挂一、揲四、归奇
- 每变一次，三变得一爻，十八变得一卦
- 起卦时全止呼吸、专念所问之事（"至诚无息"）
- 辅以"年月日时起卦法"（邵康节梅花易数变体）

## 四、每卦结构（统一模式）

每卦的解析遵循以下结构：

1. **卦名标题** — 如 "01乾为天"
2. **卦字释义** — 解字源、卦象象征
3. **卦辞** — 原文 + 高岛逐字详解
4. **《彖传》** — 孔子释卦辞 + 高岛逐句破解
5. **《大象》** — 卦象总结
6. **【占】** — 分类断语（问战征 / 营商 / 功名 / 婚姻 / 家宅 / 疾病 / 六甲等）
7. **逐爻解析**（6爻 × 每爻包含）:
   - 爻辞原文
   - 《象传》
   - 高岛详解（爻位分析、卦变分析、互卦分析）
   - 【占】（问战征 / 营商 / 功名 / 婚姻等分类断语）
   - 【例】（高岛亲身占例，含"断曰"）
8. **占例附录** — 每卦2-10+条实际占卜案例

## 五、核心思想

| 主题 | 要点 |
|------|------|
| **占卜心法** | "至诚无息"——起卦时停止呼吸、全心专注所问之事 |
| **易理本质** | 八卦=八原子（天、泽、火、雷、风、水、山、地），万物由八原子组合结晶而成 |
| **实用主义** | 反对纯义理研究，强调"照之于事物之实际"，附以经验点断 |
| **神人交通** | 通过至诚通神，感得神意于筮数，前知将来吉凶祸福 |
| **占断领域** | 战征、营商、功名、婚姻、家宅、疾病、天气、个人气运、国际关系等 |
| **解卦方法** | 卦辞 + 爻辞 + 彖传 + 象传 + 卦变 + 互卦 + 爻位 + 时运消长 |
| **阴阳消长** | 六爻示一时之缓急、事之难易，如四时循环、昼夜交替 |
| **行易三要** | 明《易》理、通世事人情、至诚通神 |

- **中村敬宇跋**: 记述高岛入狱七年、以易为学的经历。赞其"勇于为善，而臻此福报"，述其父好行善、德谊济众，因果相承。
